WEB MODULES

🧩 CUBY Platform

Live

Company admin hub: org setup, users, licenses, payments, and workspace access.

  • Create & manage companies
  • Add users and assign access
  • Create/assign licenses, view payments
  • Owner workspace switching (Project / Manufacturing)
  • Unified entry point for the ecosystem

🏭 CUBY Factory

Live

A production process builder with visual control of resource flows: define manufacturing technologies, processes, and work instructions.

  • Build process structures (stages & steps)
  • Attach instructions, checklists, and standards
  • Reuse templates across products/projects
  • Version standards for controlled change
  • Make execution repeatable and auditable

🏗️ CUBY Construction

Live

Full construction technology model: define how a house is built, break work into controllable units, and manage execution.

  • Create construction technologies with stages
  • Tasks split into work tasks and inspection tasks
  • Work tasks divided into quants (min work units: operations, criteria, instructions)
  • Shift start & shift end tasks for structured workdays
  • Role distribution: assign teams and responsibilities

📋 CUBY Project

Live

Real project creation based on Construction technology: apply a ready-made building model to a specific site and turn it into a managed project.

  • Create a project: name, address, and location on the map
  • Select which stages from the technology to execute
  • Assign people and teams to the project
  • Set up the schedule and deadlines for each phase
  • Generate quants for task distribution to the mobile app

🏠 CUBY Home Constructor

Live

Visual home designer: create your dream house in a graphic editor and launch it into production.

  • Home configuration
  • Save & compare variants
  • Generate a "starter" package for Project
  • Fast path from idea → execution

⚙️ CUBY Manufacturing

Live

Shopfloor production management: orders, readiness, and quality confirmation.

  • Manufacturing orders & work tracking
  • Real‑time readiness status for assemblies
  • Quality checkpoints and sign‑offs
  • Prepare items for packaging/shipping
  • Sync progress back to Project

🚚 CUBY Delivery

Live

Logistics layer: packing, shipment tracking, and delivery confirmation on site.

  • Pick/pack and labeling workflow
  • Shipment creation and tracking statuses
  • Proof of delivery and exception handling
  • Site receipt confirmations (scan + photos)
  • Close the loop back to Project

🤖 CUBY AI Instruction

Live

Intelligent assistance: natural language queries for technical documentation and standards.

  • Voice-activated technical support
  • Smart search across standards
  • Automated anomaly detection
  • Predictive quality guidance
